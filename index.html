Tentang Saya

Saya adalah seorang profesional yang berpengalaman dalam analisis data kuantitatif menggunakan bahasa pemrograman R, dengan kemampuan mendalam dalam membaca dan menginterpretasi scatterplot, chart, dan berbagai visualisasi data lainnya. Di sisi pengembangan web, saya menguasai teknologi frontend seperti HTML, CSS, JavaScript, Vue.js, dan React.js, serta backend dengan Express.js. Untuk pengelolaan database, saya berpengalaman dengan MySQL, Microsoft Access, MongoDB, dan Microsoft Excel. Kombinasi keterampilan analitik dan teknis ini memungkinkan saya untuk menghadirkan solusi yang komprehensif dan berbasis data.
